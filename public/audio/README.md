# Arquivos de Risadas

Este diretório deve conter os seguintes arquivos de áudio de risadas:

## Arquivos necessários:

- `laugh1.mp3` → Risada leve (para piadas sutis/inteligentes)
- `laugh2.mp3` → Risada forte (para piadas muito engraçadas)
- `laugh3.mp3` → Risada média (para piadas normais)
- `laugh4.mp3` → Risada leve alternativa (para piadas sutis)

## Classificação automática:

O sistema usa IA (Claude) para classificar cada piada e escolher a risada apropriada:

- **Rating 1** (sutil/inteligente): `laugh1.mp3` ou `laugh4.mp3` (escolha aleatória)
- **Rating 2** (muito engraçada): `laugh2.mp3`
- **Rating 3** (média/normal): `laugh3.mp3`

## Formato recomendado:

- Formato: MP3
- Duração: 2-5 segundos
- Volume: Normalizado (será ajustado automaticamente para 60% do volume da piada)

## Como adicionar:

1. Baixe ou grave os arquivos de risada
2. Coloque-os neste diretório (`/public/audio/`)
3. Certifique-se de usar os nomes exatos listados acima

## Fontes sugeridas:

- Gravar risadas reais de plateia
- Usar sons de domínio público
- Gerar com IA de áudio (se disponível)
- Bibliotecas de efeitos sonoros livres de direitos
